# SmartBrain-api

### Backend server of smart-brain app.

The docker-compose file include:
* Node express server
* Postgres database
* Redis database


### Get Started

```bash
git clone git@github.com:haochunchang/smart-brain-api.git
cd smart-brain-api
docker-compose up
```


The image recognition function uses the [Clarifai API](https://www.clarifai.com/).
